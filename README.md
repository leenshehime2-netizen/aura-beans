AuraBeans ☕

AuraBeans is a React-based coffee shop website developed as a JavaScript-to-React conversion project.

Features

- Home page with a Custom Blend Calculator
- Gears & Brews page showcasing coffee equipment
- Our Story page introducing the AuraBeans brand
- Visit Us page with location and contact information
- Responsive and user-friendly design

Technologies Used

- React.js
- JavaScript
- HTML5
- CSS3

Project Structure

- "App.js" - Main application component
- "App.css" - Application styling
- "images/" - Images and media assets

Installation

1. Clone or download the project.
2. Install dependencies:

npm install

3. Start the development server:

npm start

4. Open http://localhost:3000 in your browser.
